# 簡単なブログを作ってみる
## はじめに
この記事では、簡単なブログを作るための手順を紹介します。
今日の勉強会に関する資料はこちらになります [資料](https://www.notion.so/Next-js-9272a31bf71343459ba2d9a11c73056f?pvs=4)

## 環境
- TypeScript
- React
- Next.js

## 手順
1. プロジェクトの作成
2. ページの作成
3. 記事の作成
4. 記事の一覧表示
5. 記事の詳細表示

## ブログに関して
今回は登録フォームやログインなどの機能は実装しません。
ブログの記事に関してはマークダウンファイルを使用します。
